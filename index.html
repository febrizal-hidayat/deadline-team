Web Programming Tips & Tricks

It’s often said that there’s no substitute for experience - but there are shortcuts to it. We spoke to seasoned developers; people who toil over loops and slave over attributes day in and day out. This collection of tips comes straight from the code-face.

HTML Basics

All HTML documents must start with a document type declaration: '<''!'DOCTYPE html'>'.The HTML document itself begins with '<'html'>' and ends with '<''/'html'>'.The visible part of the HTML document is between '<'body>' and '<'/body'>'.

How to Use CSS

Use the HTML style attribute for inline styling. Use the HTML '<'style'>' element to define internal CSS. Use the HTML '<'link'>' element to refer to an external CSS file. Use the HTML element to store '<'style'>' and '<'link'>' elements. Use the CSS color property for text colors.
